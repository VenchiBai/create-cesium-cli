# create-cesium-cli 
 一款生成cesium项目的脚手架。
 集成了cesium、axios、echarts等必备开发库。

 使用方法：

 ```
 create-cesium-cli [projectName] [-vue/vue3/react]
 // [projectName]: 项目名称
 // []：默认vue3, -vue/-vue2: vue2, -react

create-cesium-cli -v, 版本
create-cesium-cli list, 支持的框架列表
```

## 版本介绍
vue3: 集成了vuex, vue-router, element-plus, vite构建优化。
