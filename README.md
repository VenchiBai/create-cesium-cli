# create-cesium-cli 
 一款生成cesium项目的脚手架。
 
 可选模板：Vue3/Vue2/React， 集成了cesium、axios、echarts，优化打包构建。

 安装：
```
npm install create-cesium-cli -g
```
 使用：

 ```
 create-cesium-cli init [vue3/vue2/react][projectName] 
```
